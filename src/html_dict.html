Dicionário de Tags HTML

Tags HTML Utilizadas

  • <!DOCTYPE html>: Declara o tipo de documento, indicando que a página está em HTML5.
  • <html>: Define o início do documento HTML.
  • <head>: Contém metadados sobre o documento, como o título e links para recursos externos.
  • <meta>: Especifica metadados como charset e viewport.
  • <title>: Define o título da página que aparece na aba do navegador.
  • <link>: Conecta recursos externos, como uma folha de estilo (CSS) ou ícone (favicon).
  • <style>: Permite a inclusão de código CSS diretamente no documento HTML.
  • <body>: Define o corpo do documento, onde o conteúdo visível é colocado.
  • <header>: Representa o cabeçalho de uma página ou de uma seção.
  • <div>: Elemento de divisão de bloco genérico para organizar o layout.
  • <nav>: Define uma seção de navegação.
  • <ul>: Define uma lista não ordenada (com marcadores).
  • <li>: Define um item de lista.
  • <a>: Cria um link para outra página ou seção da página.
  • <main>: Define o conteúdo principal do documento.
  • <aside>: Representa uma seção que está lateralmente relacionada ao conteúdo principal.
  • <section>: Define uma seção dentro do documento.
  • <h1> a <h2>: Define títulos e subtítulos.
  • <p>: Define um parágrafo de texto.
  • <span>: Usado para agrupar texto ou outros elementos inline.
  • <img>: Exibe uma imagem.
  • <footer>: Define o rodapé do documento ou de uma seção.
  • <address>: Representa informações de contato.
  • <script>: Utilizado para incorporar ou referenciar scripts JavaScript.
  • <strong>: Aplica ênfase forte ao conteúdo, geralmente exibido em negrito.
